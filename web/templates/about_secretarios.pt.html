Sobre o Banco de Dados sobre os Secretariados Estaduais

Com o objetivo de criar base de dados inéditas, reunindo informações sobre ocupantes de cargos do primeiro escalão dos governos estaduais e do Distrito Federal, foi constituída a Rede Federativa de Pesquisadores.

O trabalho nos estados concentrou-se nas informações básicas sobre os secretariados estaduais como mínimo denominador comum. As variáveis coletadas referem-se às características dos secretários estaduais entre 1995 e 2010. Os dados foram coletados, prioritariamente, por pesquisador(a) local capaz de se incorporar ao projeto.

Foram usados como fonte de identificação dos secretários diários oficiais, informações disponibilizadas pelos governos estaduais, e por outros órgãos oficiais. Para recuperação dos dados de filiação partidária foram usados como fonte informações oficiais do banco de filiados e cruzamentos com informações sobre candidatos a cargos eletivos, ambas as informações fornecidas pelo Tribunal Superior Eleitoral.

A publicização destes dados torna possível a análise dos secretariados se 26 estados além do Distrito Federal no período coberto pelo projeto a partir das características pessoais e políticas dos secretários. Estas informações foram padronizadas através do site comum da pesquisa para a entrada de dados http://ipe.cepesp.io/

No momento estamos trabalhando na atualização dos dados para o período mais recente, que se estende até 2018.

O projeto contou com apoio da FAPESP através dos projetos temáticos “Instituições políticas e gastos públicos: um estudo comparativo dos estados brasileiros” (2009 – 2013) e “As Instituições Políticas Subnacionais. Um estudo comparativo dos Estados Brasileiros” (2014 – 2020), além de apoio da Fundação Getúlio Vargas através do projeto “Visualizando a Política”.
